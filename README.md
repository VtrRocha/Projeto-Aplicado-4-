Projeto Previsão de Vendas - Amazon.

O objetivo principal é desenvolver um modelo capaz de prever vendas futuras com base em dados históricos utilizando técnicas de séries temporais.
Objetivos específicos:
•	Analisar o comportamento temporal das vendas;
•	Identificar tendências e sazonalidades;
•	Preparar e tratar dados temporais;
•	Treinar modelos preditivos;
•	Avaliar a precisão das previsões.
O projeto busca demonstrar como técnicas de previsão podem apoiar decisões estratégicas no varejo online.
